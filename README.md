#功能特性

#顾名思义，该项目属于信息管理类应用。主要功能即通过JDBC调用MySQL，完成对数据的增删改查。
#在此系统中，我们设定了面向员工及老板（管理员）两种角色的使用场景：
#1、员工可以对仓库信息进行管理。
#2、老板可以对员工个人信息及仓库信息进行管理。

#本项目使用由字符输出到控制台的方式展示菜单等信息，可以通过输入几个按键完成数据管理操作，相比纯代码操作更具友好性。
